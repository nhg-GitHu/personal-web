# 北极熊个人网站

这是用代码写出来的个人网站！

## 技术栈

- **框架**：Next.js 16 (App Router)
- **样式**：Tailwind CSS
- **内容**：MDX
- **部署**：Vercel

## 项目特点

1. 支持 **中英双语** 显示
2. **深色模式** 自动切换
3. 文章使用 MDX 编写，支持嵌入 React 组件
4. 项目板块支持 **评论互动**

## 代码示例

```tsx
export default function Hello() {
  return <h1>Hello, Polar Bear!</h1>;
}
```
