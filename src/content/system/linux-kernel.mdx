# Linux 内核入门

Linux 是一个开源的类 Unix 操作系统内核。

## 内核架构

Linux 采用**单内核架构**，但在设计上使用了模块化技术。

### 核心组件

- **进程管理**：调度和管理进程
- **内存管理**：虚拟内存和物理内存映射
- **文件系统**：VFS 抽象层支持多种文件系统
- **网络栈**：TCP/IP 协议栈实现

## 查看系统信息

```bash
# 查看内核版本
uname -r

# 查看已加载的模块
lsmod

# 查看进程
ps aux
```
